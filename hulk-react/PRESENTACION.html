HULK Interpreter

Intérprete del lenguaje HULK con interfaz web moderna

¿Qué es HULK?

HULK (Havana University Language for Kompilers) es un lenguaje de programación educativo desarrollado en la Universidad de La Habana como proyecto del curso de programación de MatCom.

Este proyecto es una reimplementación web del intérprete original, construido con React + TypeScript, que brings el lenguaje HULK al navegador con una interfaz moderna estilo chatbot.

🎯 Propósito Educativo

Diseñado para enseñar conceptos de compiladores e intérpretes

🌐 Accesible Web

Ejecuta código HULK directamente en el navegador

✨ UI Moderna

Interfaz estilo chat con diseño premium minimalista

⚡ Funcional Completo

Soporta funciones, recursión y expresiones complejas

Características del Lenguaje

Tipos de Datos

  • Números: enteros y decimales (ej: 42, 3.14)
  • Strings: texto entre comillas (ej: "Hola mundo")
  • Booleanos: true / false
  • Constantes: PI (π) y E (e)

Operadores

Aritméticos

+ - * / % ^

Comparación

== != < <= > >=

Strings

@ concatenación

Construcciones del Lenguaje

// Funciones
function fib(n) => if (n > 1) fib(n-1) + fib(n-2) else 1;
fib(10)  // 89

// Let bindings
let x = 5 in x * 2  // 10

// If expressions
if (x > 0) "positivo" else "negativo"

Funciones Built-in

  • print(expr) - Imprime y retorna el valor
  • sin(angulo) - Seno trigonométrico
  • cos(angulo) - Coseno trigonométrico
  • log(base, valor) - Logaritmo

Cómo Usarlo

1

Clona el repositorio

git clone https://github.com/leanaldrtrdz2003-byte/hulk-react.git

2

Instala dependencias

npm install

3

Inicia el servidor de desarrollo

npm run dev

4

¡Escribe código HULK!

Abre http://localhost:5173 en tu navegador

Comandos

  • /clear - Limpia el historial del REPL
  • Presiona Enter para ejecutar
